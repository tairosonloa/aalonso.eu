Bash script to remove orphans packages on ArchLinux based distros


I have created a bash script that allows the user to remove all orphans packages and their dependencies that aren't used by other packages. It only works on ArchLinux based distros, which use pacman (or pacaur) as packages manager. There is more information on the repository README file.

https://github.com/tairosonloa/remove_orphans_script
