Proyectos

Script en python 3 para descargar el contenido de los cursos de Aula Global UC3M

He hecho un script para descargar todo el contenido de los cursos de Aula Global de la UC3M, el aula virtual de mi universidad (Moode). Usa la API de Moodle para loguear un usuario, ver sus asignaturas, y descargar los ficheros dentro almacenándolos en carpetas con el nombre de la asignatura.

https://github.com/tairosonloa/Aula_Global_UC3M

Script de bash para eliminar paquetes huérfanos en distribuciones ArchLinux y derivadas

He creado un script de bash que permite al usuario eliminar todos los paquete huérfanos y sus dependencias que no son usadas por otros paquetes. Solo funciona en distribuciones ArchLinux y derivadas, que usan pacman (o pacaur) como gestor de paquetes. Hay más información en el fichero README del repositorio.

https://github.com/tairosonloa/remove_orphans_script

Script y utilidades para servidor Minecraft

Estuve administrando un servidor de Minecraft donde estaban jugando algunos amigos. He creado algunos scripts y un servicio de systemd para asegurar su disponibilidad y backups diarias. Es un código y scripts bastantes simples, pero funcionan.

https://github.com/tairosonloa/minecraft-linux-server
